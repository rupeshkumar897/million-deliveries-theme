# Million Deliveries - WordPress Custom Theme Task

This is a pixel-perfect WordPress theme development task based on the provided Figma design. The project is built with a focus on responsiveness, clean UI, and WordPress best practices.

## 🚀 Tech Stack
- **CMS:** WordPress
- **Styling:** Tailwind CSS (Responsive Design)
- **Interactivity:** Alpine.js (Mobile Menu & UI Toggles)
- **Icons:** FontAwesome
- **Fonts:** Custom Google Fonts as per Figma

## 🛠️ Features Included
- **Pixel-Perfect Hero Section:** Overlapping tracking box with high-fidelity shadows.
- **Why Choose Us Section:** Complex zig-zag layout for desktop and clean stacked layout for mobile.
- **Bento Grid:** Interactive and responsive "Comprehensive Delivery Solutions" grid.
- **Dynamic Header:** Sticky navigation with active page highlighting and a custom mobile drawer.
- **WordPress Integrated:** Uses `get_template_directory_uri()` and standard WP menu structures.

## 📦 Installation & Setup
1. Clone this repository into your `wp-content/themes/` directory.
2. Activate the theme from the WordPress Dashboard (**Appearance > Themes**).
3. Ensure you have the **FontAwesome** and **Tailwind Play CDN** scripts linked in the header (included in this version).

## 📱 Responsiveness
The theme is fully tested on:
- Desktop (1440px+)
- Tablet (768px - 1024px)
- Mobile (iPhone SE, Android - 360px+)

## 📝 Note
*Used Tailwind Play CDN for quick task demonstration. For a production-ready environment, I recommend compiling via Tailwind CLI/PostCSS for better performance.*
